API Legekrogen

Herunder er de forskellige endpoints:

  • GET og POST /api/products - Hent alle produkter og/eller opret et nyt
  • GET, DELETE og UPDATE /api/products/:id - Hent, opdater eller slet et produkt via ID
  • GET og POST /api/reviews - Hent alle reviews og/eller opret et nyt
  • GET, DELETE og UPDATE /api/reviews/:id - Hent, opdater eller slet et review via dets ID
  • POST /api/register - Opret ny bruger
